# 🧭 Campus Compass – Smart Campus Navigation App UI/UX Case Study

## Overview

Campus Compass is a smart campus navigation mobile app designed to help students easily find buildings, departments, classrooms, events, and campus services. The project focuses on improving navigation through user-centered design, clear information architecture, and an intuitive mobile experience.
